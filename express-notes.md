# Express Notes

## Endpoints


**Users**

post - signup

post - login

get - me

delete - remove account

patch - update account

**Chats**

get - my chat

post - new chat

**Connections**

get - my connections

get - search connections

post - new connection

delete - remove connection

**Messages**

get - chat messages

post - new message

delete - delete message (stretch)

patch - edit message (stretch)
